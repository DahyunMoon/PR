make 방법: make clean -> make main 순서대로 실행하면 됩니다
debug 방법: make clean -> make main_debug 순서대로 실행하면 됩니다
실행파일 실행방법: ./main을 실행하면 됩니다
메인메뉴 설명
1. create_sche: 새로운 과목 스케줄을 추가합니다
2. read_sche: 현재 과목 스케줄에서 과목 하나의 스케줄을 읽어옵니다
3. update_sche: 현재 과목 스케줄에서 과목 하나의 스케줄을 업데이트합니다
4. delete_sche: 현재 과목 스케줄에서 과목 하나의 스케줄을 삭제합니다
5. update_sches: 현재 과목 스케줄에서 일정 조건으로 묶인 과목들의 스케줄을 업데이트합니다
6. delete_sches: 현재 과목 스케줄에서 일정 조건으로 묶인 과목들의 스케줄을 삭제합니다
7. list_sche: 현재 관리되고 있는 모든 과목 스케줄을 읽어옵니다
8. search_name: 현재 과목 스케줄에서 과목 이름을 통하여 해당 과목 스케줄을 검색합니다 
9. search_major: 현재 과목 스케줄에서 전공 번호를 통하여 해당 과목들의 스케줄을 검색합니다
10. search_code: 현재 과목 스케줄에서 과목 번호를 통하여 해당 과목 스케줄을 검색합니다
11. search_diff: 현재 과목 스케줄에서 전공별 난이도를 통하여 해당 과목들의 스케줄을 검색합니다
12. sort_score: 현재 과목 스케줄에서 과목별 점수에 따라서 과목별로 우선순위를 부여하여 정렬합니다
13. read_file: 과목 스케줄의 데이터 파일을 읽어옵니다 
    read_report: 과목 스케줄의 보고서 파일을 읽어옵니다
14. save_file: 과목 스케줄의 데이터 파일을 저장합니다
    save_report: 과목 스케줄의 보고서 파일을 저장합니다
15. debug_records: 과목 스케줄이 잘 저장되고 있는지 debugging을 통해 확인합니다
16. load_file: 전체 50개 과목의 스케줄의 데이터 파일을 읽어옵니다
